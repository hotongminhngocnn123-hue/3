## 📌 GIỚI THIỆU SẢN PHẨM

Hệ thống **Quản lý Sản phẩm** là một phần mềm được xây dựng nhằm hỗ trợ quản lý dữ liệu sản phẩm một cách:

- Nhanh chóng  
- Chính xác  
- Khoa học  

Chương trình được phát triển bằng:

- 🐍 Ngôn ngữ **Python**
- 🖥 Thư viện **Tkinter** (thiết kế giao diện)
- 📊 Thư viện **Pandas** (xử lý và lưu trữ dữ liệu dạng Excel)

Phần mềm được lập trình và chạy trên môi trường **Visual Studio Code**.

---

## 👥 ĐỐI TƯỢNG SỬ DỤNG

- Doanh nghiệp nhỏ và vừa  
- Cửa hàng kinh doanh sản phẩm  
- Nhân viên quản lý kho  
- Người có trình độ tin học cơ bản  

Phần mềm được thiết kế với giao diện đơn giản, dễ thao tác, nên người dùng chỉ cần kỹ năng máy tính cơ bản là có thể sử dụng hiệu quả.

---

## ⭐ GIÁ TRỊ MANG LẠI

- Quản lý danh sách sản phẩm rõ ràng, có hệ thống  
- Giảm sai sót trong tính toán nhờ tự động hóa  
- Tiết kiệm thời gian nhập và tìm kiếm dữ liệu  
- Lưu trữ thông tin an toàn, lâu dài  

---

## 🛠 CÔNG NGHỆ SỬ DỤNG

| Công nghệ | Mục đích |
|-----------|----------|
| Python | Ngôn ngữ lập trình chính |
| Tkinter | Thiết kế giao diện |
| Pandas | Xử lý và lưu trữ dữ liệu |
| Excel | Lưu file dữ liệu |
| Visual Studio Code | Môi trường phát triển |
